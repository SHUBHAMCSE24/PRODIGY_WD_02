# PRODIGY_WD_02

## Stopwatch Web Application

This project is a fully functional Stopwatch Web Application developed using HTML, CSS, and JavaScript as part of the Web Development Internship at Prodigy Infotech.

## Features

- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Lap Time Recording
- Milliseconds Support
- Responsive Design
- Modern User Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```bash
PRODIGY_WD_02
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run the Project

1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html` using Live Server
